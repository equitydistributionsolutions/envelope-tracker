❓ How it works
🔗
5 named links per distribution: Dashboard (= the GitHub Pages tracker), Project Folder (Google Drive), Tracker Sheet (digital envelopes), IRS Tracker (mail / IRS), and IRS Cover Letter folder. The mail icon opens a pre-filled email with all five links.
📊
Live completion from Google Sheets: When the Tracker Sheet URL and IRS Tracker URL are entered, this hub fetches live row data. Digital counts "Completed" rows; Mail counts rows with a date in "Mail Date". Fetches on page load and whenever you click Refresh.
💰
Profit: Cost = Rate × Envelope count. Profit = Revenue − Cost. The profit chart and stats update as you add data.
♻️
Export / Import to back up to a file or sync data across computers.
